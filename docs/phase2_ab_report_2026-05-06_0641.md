### Performance 噪声分析

Phase 2 v=2 显示 PASS rate -25% / cross-dataset rate -34% vs v=1。**这不是 Phase 2 设计缺陷的证据**，原因：

1. **样本量不足**：v=1 effective n=3 (2 COMPLETED + 1 RUNNING)，v=2 effective n=3。
   每组 13-14 PASS / 163-239 FAIL — Bayesian posterior 在 N=3 task 下 power 极低
2. **Plan v5+ §V-1 灰度 criterion**：PASS rate within 30% margin or improved。**v=2 -25% 仍在容忍范围内**
3. **F-5 第二轮决议**：要求 N≥12 task 才进 "trial" stage，N≥20 才出 "verdict"
4. **混淆变量**：B11 期间 KB 同时被 v=1/v=2 写入；KB cold start 行为可能影响早期 round（hypothesis-keyed 检索还没积累足够数据就被采用）
